# Telesto (Accommodations telesto style)

> Used in [Drenthe Campings](https://www.drenthecampings.nl/nl/camping-kiezer)

Example:
```html
<div is="telesto"></div>
```

Parameters:

#### category-id

The category to be shown.

number | default value: `null`

```html
<div is="telesto" :category-id="1"></div>
```
