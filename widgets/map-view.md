# Map view

Example:
```html
<div is="map-view"></div>
```

Parameters:

#### category-id

Load the provided category.

number | default value: `null`

```html
<div is="map-view" :category-id="1"></div>
```

#### all

Load all categories from harbor.

number | default value: `true`

```html
<div is="map-view" :all="false"></div>
```
